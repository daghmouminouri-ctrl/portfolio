# Portfolio Tracker

An Excel-based portfolio tracker for monitoring holdings, allocation, gains/losses, and top positions across stocks, ETFs, and alternative assets. The workbook contains a main portfolio overview sheet, with summary metrics including total market value, total cost, total gain/loss, total return, asset allocation, and top 5 holdings.

## What this workbook does

The tracker records each position with fields for ticker, name, shares, cost basis, current price, market value, total cost, gain/loss, return percentage, and portfolio weight. It groups holdings into three buckets — stocks, ETFs, and alternative assets — and rolls them into a portfolio total section plus a summary area for allocation and concentration analysis.

## Workbook structure

### Sheet1

It includes the full position list, portfolio totals, asset-class allocation, and a ranked top-5 holdings section, making it the primary operating sheet for updates and review.

## Main metrics tracked

- Total market value of the portfolio
- Total cost basis across all holdings
- Absolute gain/loss
- Total portfolio return percentage
- Number of holdings
- Asset allocation by category: stocks, ETFs, and alternatives 
- Top 5 holdings by amount

## Asset categories currently used

| Stocks | Individual equity positions such as NVIDIA, Apple, Microsoft, Alphabet, Coca-Cola, PepsiCo, Meta, Amazon, Waste Management, Hormel, Nike, ASML, TELUS, Duke Energy, and Enbridge 
| ETFs | Broad or thematic fund exposure, including QQQ-related and S&P 500 positions
| Alternative | Non-core diversifiers such as gold, bitcoin exposure, and other non-traditional positions 

## How to use it

1. Update the latest prices or refresh the input data feeding the current price column.
2. Add or edit positions in the holdings table by adjusting ticker, shares, and cost basis fields as needed.
3. Review the summary block to monitor total return, allocation drift, and concentration in the top holdings.
4. Use the ranking area and allocation section to spot overweights, underweights, and whether the portfolio is becoming too dependent on a few names.

## Why this tracker is useful

This workbook gives a compact view of both performance and risk concentration in a single file. The combination of position-level return data, category allocation, and top-holdings ranking makes it practical for periodic rebalancing and for checking whether the portfolio still matches the intended strategy.
